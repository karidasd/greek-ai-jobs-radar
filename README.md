## Τι είναι αυτό;

Ένα **100% δωρεάν, open-source εργαλείο** που κάθε βράδυ σαρώνει αυτόματα την αγορά εργασίας στην Ελλάδα και εμφανίζει:

- 🇬🇷 **Live αγγελίες** από Ελληνικές εταιρείες (Skroutz, Novibet, Blueground, Orfium, Spotawheel...)
- 💰 **Πραγματικούς μισθούς** — Μικτά & Καθαρά, σε Ευρώ, με φόρους
- 📈 **Τάσεις αγοράς** — Ποιες τεχνολογίες ζητούνται περισσότερο (Python, SQL, AWS...)
- 🛑 **Υπολογιστή Αδικίας** — Δες αν πληρώνεσαι λιγότερο από τον μέσο όρο
- 🎯 **Skill-Sniper** — Βάλε τα skills σου και δες ποιες αγγελίες σου ταιριάζουν
- 📡 **Graphical Radar** — Live scanner με τοποθεσίες αγγελιών

---

## 🛡️ Γιατί δεν ξύνουμε LinkedIn & Kariera.gr;

Απλό: **Cloudflare + Anti-bot = 403 Forbidden** σε κάθε αυτοματοποιημένο εργαλείο.

Αντί γι' αυτό, ανακαλύψαμε ότι όλες οι μεγάλες Ελληνικές Tech εταιρείες χρησιμοποιούν το **Workable** ως σύστημα HR. Το Workable έχει ανοιχτό JSON API:

```
POST https://apply.workable.com/api/v3/accounts/{company}/jobs
```

Το καλούμε απευθείας → παίρνουμε **καθαρά δεδομένα** χωρίς bot detection, χωρίς κόστος.

## ⚖️ Τα Δικαιώματά σου — Οδηγία ΕΕ 2023/970 (από 1/7/2026)

> **Αν η εταιρεία δεν γράφει μισθό στην αγγελία, παρανομεί.**

Η [Οδηγία 2023/970 της ΕΕ](https://eur-lex.europa.eu/legal-content/EL/TXT/?uri=CELEX%3A32023L0970) για τη **Μισθολογική Διαφάνεια** ισχύει από 1η Ιουλίου 2026. Υποχρεώνει κάθε εταιρεία να αναγράφει τον μισθό (ή εύρος μισθού) σε κάθε αγγελία — **πριν** την πρώτη συνέντευξη.

### Τι δικαιούσαι:

| Δικαίωμα | Τι σημαίνει στην πράξη |
|----------|------------------------|
| 📄 **Μισθός στην αγγελία** | Υποχρεωτική αναγραφή εύρους μισθού πριν τη συνέντευξη |
| 🙊 **Δεν αποκαλύπτεις εσύ πρώτος** | Η εταιρεία δεν επιτρέπεται να ρωτήσει τον τρέχοντα μισθό σου |
| 📊 **Δικαίωμα σύγκρισης** | Μπορείς να ζητήσεις γραπτώς τον μέσο μισθό συναδέλφων ίδιας θέσης |
| ⚖️ **Ίση αμοιβή** | Ίδια εργασία = ίδιος μισθός, ανεξαρτήτως φύλου ή υπηκοότητας |
| 🚫 **Απαγόρευση αντιποίνων** | Δεν επιτρέπεται απόλυση / υποβάθμιση λόγω άσκησης αυτών των δικαιωμάτων |
| 📬 **Καταγγελία** | Αν παρανομούν → [ΣΕΠΕ](https://www.sepe.gr) (Σώμα Επιθεώρησης Εργασίας) |

### Τι κάνεις όταν δεν υπάρχει μισθός στην αγγελία:

**1.** Στείλε email πριν τη συνέντευξη:
```
"Σύμφωνα με την Οδηγία ΕΕ 2023/970 για τη Μισθολογική Διαφάνεια,
παρακαλώ γνωστοποιήστε το εύρος αποδοχών για τη θέση πριν προχωρήσουμε."
```

**2.** Αν αρνηθούν → Καταγγελία στη **ΣΕΠΕ**: [sepe.gr/kataggelies](https://www.sepe.gr)

**3.** Μη πας σε συνέντευξη χωρίς να ξέρεις τον μισθό. Αυτό είναι **νόμιμο δικαίωμά σου**.

> Μια εταιρεία που κρύβει τον μισθό, τον κρύβει γιατί ξέρει ότι δεν είναι ανταγωνιστικός.

### Βήμα 1: Εξαγωγή Μισθού (NLP Regex)
Ψάχνουμε στην περιγραφή κάθε αγγελίας για salary patterns:
- `$150k` → USD → EUR (x 0.92)
- `£50k` → GBP → EUR (x 1.17)
- `2.500€/μήνα` → x14 μηνιάτικα = Ετήσιος Μισθός

### Βήμα 2: Καθαρός Μισθός ανά Χώρα

| Χώρα | Διαίρεση | Κρατήσεις |
|------|----------|-----------|
| 🇬🇷 Ελλάδα (<20k€) | ÷ 14 | ~25% |
| 🇬🇷 Ελλάδα (<40k€) | ÷ 14 | ~30% |
| 🇬🇷 Ελλάδα (>40k€) | ÷ 14 | ~35% |
| 🇪🇺 Ευρώπη & UK | ÷ 12 | ~40% |
| 🌎 Worldwide / USA | ÷ 12 | ~30% |
